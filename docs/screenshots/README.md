# Screenshot Guidelines

Screenshots are welcome for the README and GitHub releases, but they must not
expose private information.

Before publishing a screenshot:

- Do not show real Windows usernames.
- Do not show personal file paths.
- Do not show private documents, downloads, photos, projects, or filenames.
- Do not show email addresses, tokens, license keys, or account names.
- Prefer a clean test Windows profile or mock folders created only for screenshots.
- Crop or blur anything that could identify a person, device, company, or private project.

Recommended screenshot set:

- Dashboard
- Temp Cleaner preview
- Startup Analyzer results
- Disk Analyzer Lite results
- Activity Log
